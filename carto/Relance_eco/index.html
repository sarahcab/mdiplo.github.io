Les boutons permettent de faire circuler les flux financiers. Les fonds en bleus et les pièces mettrent le capital disponible dans les noeuds : l'Etat peut continuer d'alimenter le système lorsque le solde est négatif, ce qui n'est pas le cas des entreprises ou des ménages.
Les dessins en noir représentent les acquis : ils apparaissent au fur et à mesure que l'argent circule.

Cliquer sur le bouton démo pour lancer une explication. Le deuxième bouton permet d'activer le schéma.
